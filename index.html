Setting up dartlang on your system

  • ArchLinux : Install using AUR yaourt dart
  • Ubuntu linux: Follow Guide here: https://www.dartlang.org/install/linux
  • Windows: Install using Chocolatey Package Manager: cinst dart-sdk
  • macOS: Install using Homebrew: 
    $ brew tap dart-lang/dart
$ brew install dart

Tested on

  1. Windows 10.
  2. macOS El Capitan
  3. Arch Linux (Bleeding Edge)
  4. Ubuntu 16.10

Documentation

Documentation is generated using dartdoc (https://github.com/dart-lang/dartdoc)

To generate documentation for project

  • run dartdoc
  • documentation will be available in doc folder in the project root

Running the Project

Setting Up

  1. Clone the project.
  2. Navigate to project root.
  3. Run pub get

Generating Random Input in csv Files

  1. Run dart bin/generateItemCSV.dart <boycount> <girlcount> <giftcount>

Third Question

  1. Run dart bin/third.dart <girlcount> <boycount>

Libraries

Algorithm

This module contains various Algorithms to make couples

Boy

This module contains the Class to Represent a Boy and related objects

BoySearch

This module facilitates searching a boy by name using various Algorithms

Couple

This library contains Class for Couple and wrapper for ListOfCouples

Gift

This library contains Gift Class and related wrappers for List implementation

GiftSelector

This module contains various implementations for selection of Gifts

Girl

This module contains the Class to Represent a Girl and related objects

Person

This module contains class to represent a person

RandomReturn

This module facilitates Random Selection from a List

TopKPicker

This module facilitates top k selection from feasible items

Util

Util Library to abstract out basic file I/O for the project and easy Logging
